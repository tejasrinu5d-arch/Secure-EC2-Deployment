# Secure EC2 Deployment – Project Report



## Project Summary



This project demonstrates the deployment of a secure Amazon EC2 instance on AWS using Amazon Linux 2023. The implementation includes secure SSH access, Apache web server deployment, Amazon EBS storage management, snapshot creation, Golden AMI creation, and Amazon CloudWatch monitoring.



## AWS Services Used



- Amazon EC2

- Amazon EBS

- Amazon Machine Image (AMI)

- Amazon CloudWatch

- IAM

- Security Groups

- Amazon VPC



## Major Tasks Performed



- Launched an EC2 instance

- Connected using SSH

- Installed Apache Web Server

- Hosted a custom web page

- Created and attached an EBS volume

- Formatted and mounted the EBS volume

- Configured persistent storage using /etc/fstab

- Created an EBS Snapshot

- Created a Golden AMI

- Verified a new EC2 instance from the Golden AMI

- Monitored the instance using CloudWatch



## Outcome



Successfully completed a secure EC2 deployment following AWS best practices while implementing storage, backup, monitoring, and image creation features.
